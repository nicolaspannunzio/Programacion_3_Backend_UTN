# Programación III - UTN - Backend en Java

Repositorio dedicado a los Trabajos Prácticos de la segunda etapa de la asignatura Programación III de la Tecnicatura Universitaria en Programación (UTN), enfocados íntegramente en el desarrollo Backend con Java.

## Trabajos Prácticos

* **TP5 - Colecciones y Sobreescritura de Métodos:**
  * Modelado de entidades a partir de diagramas UML.
  * Implementación de relaciones de herencia y composición.
  * Uso de colecciones de tipo `Set` para garantizar la unicidad de elementos.
  * Sobreescritura profunda de los métodos `toString()`, `equals()` y `hashCode()`.

## Tecnologías Utilizadas

* **Java** (JDK 17+)
* **Apache NetBeans IDE**
* **Git & GitHub**

## Estructura del Proyecto

El proyecto está diseñado bajo el paradigma de la Programación Orientada a Objetos (POO), encapsulando la lógica en paquetes específicos. El paquete base `entidades` contiene:
* Clases abstractas (`Base`) para herencia de atributos comunes.
* Clases concretas (`Usuario`, `Pedido`, `Producto`, etc.).
* Enumeraciones (`Rol`, `Estado`, `FormaPago`) para integridad de datos.
